# MusicBot

### Descrição

MusicBot é um bot com tema musical, que recomenda artistas, álbuns, fala sobre curiosidades e outros tópicos do assunto!

### Como Rodar

Instale a extensão "Live Server" na loja do VSCode e clique no botão "Go Live" na barra inferior.

### Comandos e Palavras Chave

- "Olá"
- "você"
- "favorito"
- "música"
- "álbum"
- "artista"
- "banda"
- "Ouvir música"
- "curiosidade"
